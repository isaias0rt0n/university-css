Front-end

HTML, CSS e JavaScript são a base de toda a web. Tudo o que você está vendo aqui agora depende deste tripé. Desenvolva sites e webapps com HTML, CSS e JavaScript. Aprenda as boas práticas e as últimas versões do JavaScript. Use também framewokrs de desenvolvimento de interfaces como o Reactjs e Angular.

Back-end

O Back End trabalha em boa partes dos casos fazendo a ponte entre os dados que vem do navegador rumo ao banco de dados e vice-versa, sempre aplicando as devidas regras de negócio, validações e garantias em um ambiente onde o usuário final não tenha acesso e possa manipular algo. Venha se tornar um dev back-end.

Full-stack

Há cada vez mais demanda por desenvolvedores multidisciplinares e com alta capacidade de adaptação. Um profissional que domina tanto a camada de back-end como a de front-end, numa atuação mais generalista e menos especialista: o Full Stack Developer. Seja um desenvolvedor Full-stack e atue em várias partes de uma aplicação web.
